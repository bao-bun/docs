## AFK command
<CodeGroup>
```javascript Syntax
bao afk [reason]
```

```javascript Example 1
bao afk
```

```javascript Example 2
bao afk I am sleeping ðŸ˜´
```
</CodeGroup>

If someone mentions you, they will see how long you've been AFK for, and for what reason.
<img src="/images/afk_ping.png"/>


Now by default, if you send a message in any channel, the bot will remove your AFK status.
If you want to disable this behaviour, you can use the auto-afk command.

<CodeGroup>
```javascript Syntax
bao autoafk
```
</CodeGroup>

Then to remove your AFK status if you disable auto-afk, you can use the `bao afk` command again.

<img src="/images/auto_afk.png"/>
