### Starting a giveaway
If no channel is provided, it will default to the current one.

<CodeGroup>
```javascript Syntax
bao gw start <duration> <winners> <preset> [channel] <prize>
```

```javascript Example 1
bao gw start 12h 1w #giveaways Year of Nitro Basic
```


```javascript Example 2
bao gw start 7d 10w preset-3 Month of Nitro
```
</CodeGroup>


### Ending or rerolling giveaways 
You can end a giveaway early or reroll the winners of a giveaway that has ended.

If you need help with getting the Message ID, visit [this page](/resources/message_id).

<CodeGroup>
```javascript Syntax
bao gw end <message_id>
bao gw reroll <message_id>
```

```javascript Example
bao gw end 1346616792751149077
bao gw reroll 1567991258432115349
```
</CodeGroup>
