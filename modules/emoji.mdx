<Warning>
`<>` means an argument is required, `[]` means it's optional
</Warning>


### Emoji Enlarge
Enlarge provided emojis

<CodeGroup>
```js Syntax
/emoji enlarge [emojis]
```
</CodeGroup>
- `emojis` (text): required <br/>
  The emojis you want to enlarge.

### Emoji Steal
Steal provided emojis

<CodeGroup>
```js Syntax
/emoji steal [emojis]
```
</CodeGroup>
- `emojis` (text): required <br/>
  The emojis you want to steal.
