### Enabling DM on Join
Enable or disable DM on Join.

<CodeGroup>
```javascript syntax
/dm enable
````
</CodeGroup>

*No parameters*

### Setting DM content
Set the content used for DM on Join messages.

<CodeGroup>
```javascript syntax
/dm content <content_name>
```
</CodeGroup>
