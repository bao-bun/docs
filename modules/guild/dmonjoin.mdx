title: "DM on Join"
icon: "message"
description: "DM on Join privately messages new members who just joined the server."
---

<Warning>
`<>` means required, `[]` means optional
</Warning>

### Enabling DM on Join

Enable or disable DM on Join.

<CodeGroup>``` syntax /dm enable ````</CodeGroup>

_No parameters_

### Setting DM content

Set the content used for DM on Join messages.

<CodeGroup>
``` syntax
/dm content <content_name>
```
</CodeGroup>

- `content_name` (text) — required
  The content name you want to use for DM on Join messages.

### Setting DM delay

Set the delay for DM on Join messages.

<Note>Setting a delay of 0s sets no delay for the DM on join message</Note>

<CodeGroup>
``` syntax
/dm delay <delay>
```
</CodeGroup>

- `delay` (time) — required
  The delay duration you want to delay DM on Join messages.

### Test sending

Run a test DM for DM on Join.

<CodeGroup>``` syntax /dm test ```</CodeGroup>

_No parameters_
