<Warning>
`<>` means required, `[]` means optional
</Warning>

### Set giveaway embed
Set the embed content used for giveaways.

<CodeGroup>
```javascript syntax
/gw set-gw-embed <content_name>
```
</CodeGroup>

- `content_name` (text) — required
  The custom content name to use as the giveaway embed.

### Set reaction
Set the reaction emoji used for giveaways.

<CodeGroup>
```javascript syntax
/gw set-reaction <reaction>
```
</CodeGroup>

- `reaction` (text) — required
  The emoji used for giveaway reactions.
