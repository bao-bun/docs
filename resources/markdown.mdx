### Markdown for giveaway embed
Here is the markdown you can use for the a custom giveaway embed:
- `[host]` giveaway hoster
- `[winner_count]` amount of winners
- `[timestamp]` timestamp of when the giveaway ends
- `[prize]` giveaway prize

### Markdown for ping on join & DM on join
Here is the markdown you can use for the ping on join and the DM on join features:
- `[user_name]` username of the user
- `[user_display_name]` display name of the user
- `[user_mention]` mentions the user
- `[server_name]` name of the server
