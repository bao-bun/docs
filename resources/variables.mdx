---
title: "Variables"
icon: "code-bracket"
description: "All available variables that can be used in custom content."
---

<Tabs>
  <Tab title="Guild">
    | Variable | Output |
    |---------|--------|
    | `[server.name]` | The server’s name |
    | `[server.id]` | The server’s ID |
    | `[server.owner]` | The server owner’s username |
    | `[server.owner_mention]` | Mentions the server owner |
    | `[server.member_count]` | Total number of members |
    | `[server.boost_count]` | Number of server boosts |
  </Tab>

  <Tab title="User">
    | Variable | Output |
    |---------|--------|
    | `[user.name]` | The user’s username |
    | `[user.display_name]` | The user’s display name |
    | `[user.id]` | The user’s ID |
    | `[user.mention]` | Mentions the user |
    | `[user.avatar]` | The user’s avatar URL |
    | `[user.joined_at]` | When the user joined |
    | `[user.join_position]` | The user’s join position |
    | `[user.boosting]` | Whether the user is boosting |
  </Tab>

  <Tab title="Channel">
    | Variable | Output |
    |---------|--------|
    | `[channel.name]` | The channel’s name |
    | `[channel.id]` | The channel’s ID |
    | `[channel.mention]` | Mentions the channel |
    | `[channel.topic]` | The channel’s topic |
  </Tab>

  <Tab title="Giveaway">
    | Variable | Output |
    |---------|--------|
    | `[gw.prize]` | Giveaway prize |
    | `[gw.reaction]` | Giveaway reaction | 
    | `[gw.winners]` | Number of winners |
    | `[gw.host]` | Giveaway host |
    | `[gw.timestamp]` | Giveaway end timestamp |
    | `[gw.entry_mode]` | Entry method |
    | `[gw.host]` | Host's mention |
    | `[gw.claim_time]` | Winner's claim time
  </Tab>
</Tabs>
